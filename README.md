# React Testing Library

# Contexto

Praticando testes em react utilizanto RTL
## Técnologias usadas

 React, RTL, CSS3, HTML5, ES6

## Instalando Dependências

```
cd react-testing-library/
npm install
``` 
## Executando aplicação

  ```
    cd src/ && npm start
  ```
  
  ## Executanto testes
  
  ```
  npm test
  ```
